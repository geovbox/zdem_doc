7.11 imple¶

说明

imp[le] 设置挤压距离

使用方法

IMPLE WALL ID <int> xmove <float> print <float> ps <float> save <float>
实例

#让id=1的墙，沿着x正方向推进16000.0，每挤压2000.0保存一次计算结果
IMPLE wall id 1 xmove 16000.0 print 2000.0 ps 2000.0 save 2000.0
